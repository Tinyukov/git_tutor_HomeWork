## Списки

Чтобы добавить ненумерованные списки ,необходимо пункты выделить звездочкой (*).
Например вот так : 
* Элемент 1
* Элемент 2 

Чтобы добавить нумерованные , необходимо их просто пронумеровать.
Например вот так: 
1. Первый пункт 
2. Второй пункт
